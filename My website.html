Learn the basics of HTML, its uses, and how to create structured web pages.

What is HTML?

HTML is an acronym which stands for Hyper Text Markup Language which is used for creating web pages and web applications. Let's see what is meant bt Hypertext Markup Language and Web page.
Hyper Text : Hyper Text Simply means "Text within Text". A text has a link within it, is a hypertext . Whenever you click on a link which brings you to a new webpage, you have clicked on a hyper. HyperText is a way to link two or more web pages (HTML document ) with each other.
Markup language: A markup language is a computer language that is used to do apply layout and formatting convention to a text document. Markup language makes text more interactive and dynamic . It can turn text into images , table , link etc .

Uses of HTML

  • Structuring Web Pages
  • Embedding Images and Media
  • Creating Forms and Tables
  • External Resources
  • Displaying Media
  • Accessibility and SEO
  • Creating Links and Navigation

What does HTML do?

  • Defines the content: It organizes text, images, links, videos, and more into a readable format.

  • Creates structure: It uses tags like <header>, <footer>, and <section> to arrange content.

  • Links web pages: HTML allows you to connect multiple pages using hyperlinks.

  • Supports multimedia: It embeds images, videos, and audio files.

    Father of HTML


    Tim Berners-lee

    • Biography of Tim Berners-Lee

    • Full Name: Sir Timothy John Berners-Lee
    • Born: June 8, 1955 (age 69)
    • Birthplace: London, England
    • Nationality: British
    • Profession: Computer Scientist, Engineer, Professor

  • Tim Berners-Lee is a British computer scientist best known for inventing the World Wide Web (WWW).
  • In 1989, while working at CERN (the European Organization for Nuclear Research), he proposed a system for sharing information using hypertext.
  • In 1991, he developed HTML (HyperText Markup Language), HTTP (HyperText Transfer Protocol), and the first web browser, making the WWW accessible.
  • He was knighted by Queen Elizabeth II in 2004 for his contributions to technology.
  • Today, he advocates for an open and free internet, promoting data privacy and decentralization through projects like Solid.

    • History of HTML

  • 1. The Early Days (1989 – 1993)

    • 1989: HTML was invented by Tim Berners-Lee, a physicist at CERN (European Organization for Nuclear Research), as a way to share and link documents over the internet. 1990: The first version of HTML was written by Berners-Lee. It had only 18 tags. 1991: The first web page was published, providing information about the World Wide Web project. 1993: Marc Andreessen and Eric Bina created Mosaic, the first graphical web browser, making the web more accessible.

  • 2. HTML 2.0 (1995)

    • The first official HTML standard. Introduced new elements:
    for user input. for images. for tabular data. This version set the foundation for basic web design.

  • 3. HTML 3.2 (1997)

  • Published by the W3C (World Wide Web Consortium).
  • Added
  • <div> for better content structuring.
  • <font> for text styling (later deprecated).
  • <script> for client-side scripting.
  • This version aligned with the rise of CSS for styling.

  • 4. HTML 4.01 (1999)

  • A major release with improved semantic elements.
  • Added:
  • <abbr>, <acronym>, <strong>, and <em> for better accessibility and meaning.

  • 5. XHTML 1.0 (2000)

  • XHTML was a stricter, XML-based version of HTML. Enforced rules like: All tags had to be closed. Attributes had to be in lowercase. Proper nesting of elements was mandatory. Aimed at making web pages more consistent and portable.

  • HTML5 (2014)

  • A major overhaul introduced by the WHATWG (Web Hypertext Application Technology Working Group). Included: Native support for audio and video: <audio> and <video>. Canvas and <svg> for graphics and animations. New semantic tags: <article>, <section>, <header>, <footer>, <nav>. Better support for web applications and APIs. Improved compatibility with mobile devices.

  • 7. HTML5.1 and Beyond (2016 – Present)

  • HTML5.1 (2016) and HTML5.2 (2017) introduced small updates: <dialog> for creating modal dialogs. <picture> and <srcset> for responsive images. Ongoing improvements continue under WHATWG. Modern HTML is now living standard, with continuous updates.
